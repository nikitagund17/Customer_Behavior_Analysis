# 📊 Customer Shopping Behavior Analysis

## Overview

Customer Shopping Behavior Analysis is an end-to-end Data Analytics project that analyzes customer purchasing patterns using transactional data. The project demonstrates the complete analytics workflow, including data loading, exploratory data analysis (EDA), data cleaning, SQL-based business analysis, interactive dashboard creation in Power BI, report writing, and presentation development using Gamma AI.

The objective is to transform raw customer data into meaningful business insights that support better decision-making.

# 🚀 Project Workflow

## 1. Data Loading

* Imported the dataset using Python (Pandas).
* Examined the dataset structure and data types.
* Generated summary statistics.

## 2. Exploratory Data Analysis (EDA)

* Analyzed customer demographics.
* Identified missing values.
* Explored purchase patterns.
* Created visualizations to understand trends.
* Studied customer behavior across different categories.

## 3. Data Cleaning

* Handled missing values.
* Renamed columns for better readability.
* Removed unnecessary columns.
* Performed feature engineering.
* Standardized the dataset for analysis.

## 4. SQL Analysis

The cleaned dataset was imported into PostgreSQL, MySQL, or SQL Server to perform business-focused SQL queries, including:

* Revenue by Gender
* Customer Segmentation
* Top Rated Products
* Subscriber vs Non-Subscriber Analysis
* Discount Usage Analysis
* Revenue by Age Group
* Shipping Type Comparison
* Product Performance Analysis
* Repeat Customer Analysis

## 5. Power BI Dashboard

Designed an interactive dashboard to visualize key business metrics and insights through charts, KPIs, filters, and slicers.


## 6. Report & Presentation

Prepared a detailed project report and created a professional presentation using Gamma AI to summarize findings and recommendations.
